# Overview

Coastal Collective has built a beautiful lifestyle brand across luxury coastal stays, wellness retreats, and curated local experiences. The next opportunity is to make the website feel as considered, premium, and easy to use as the real-world experience.

This proposal outlines three ways Akima Studio can help improve Coastal Collective's digital presence, with a focus on clearer storytelling, stronger mobile UX, and a smoother path from inspiration to direct booking.

# Goals

- Improve the clarity and quality of the website experience
- Increase direct bookings and reduce reliance on Instagram enquiries
- Make the mobile journey easier for guests browsing and booking on the go
- Better communicate the premium nature of the accommodation and experiences
- Strengthen trust through clearer content, imagery, reviews, and booking prompts
- Create a more scalable website foundation for future retreats, locations, and partnerships

# Optional add-ons

# FAQs

## Can we start with the audit and decide on design later?

Yes. Option 01 is designed to provide clarity before committing to a larger redesign. If you move into a larger sprint afterwards, we can use the audit findings as the foundation for the next phase.

## Does this include development?

Not by default. This proposal focuses on strategy, UX, and design. Akima Studio can either work with your existing developer or provide a separate development estimate if needed.

## What do you need from us to get started?

We would need access to the current website, analytics if available, brand assets, imagery, booking platform details, and any existing content or customer insights.

## Which option do you recommend?

Option 02 is the recommended starting point because it gives Coastal Collective a stronger website experience and improved conversion pathway without overextending the initial scope.
